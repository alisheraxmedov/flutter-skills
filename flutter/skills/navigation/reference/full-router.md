# Full router setup

```yaml
dependencies:
  go_router: ^15.0.0
dev_dependencies:
  go_router_builder: ^3.0.0
  build_runner: ^2.4.0
```

```dart
final rootNavigatorKey = GlobalKey<NavigatorState>();

final router = GoRouter(
  navigatorKey: rootNavigatorKey,
  initialLocation: '/home',
  routes: $appRoutes, // generated by go_router_builder
  redirect: _authRedirect,
  refreshListenable: authNotifier,
  errorBuilder: (context, state) => ErrorScreen(state.error),
);

MaterialApp.router(routerConfig: router);
```

## Pieces it ties together
- `rootNavigatorKey` — lets full-screen routes (login/onboarding) cover the shell.
- `$appRoutes` — generated from `@TypedGoRoute` / `@TypedStatefulShellRoute` annotations (see `type-safe-routes.md`, `shell-routes.md`).
- `redirect` + `refreshListenable` — centralized auth gating that re-runs on auth changes (see `auth-redirect.md`).
- `errorBuilder` — friendly fallback for unmatched paths.

Run code generation after editing route classes:

```bash
dart run build_runner build -d
```
